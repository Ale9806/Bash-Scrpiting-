## OUTPUT
**Print in terminal:**
```` bash
echo "Hellow world"
````
## RIDECRECTING STREAMS: 

**Save stream ( in this case the echo output) in a file (or create a new file and save output if file is nonexisting):**
```` bash
echo "Hellow world" > file.txt
````

**Append to a file:**
```` bash
echo "Hellow world" >> file.txt
````
